## О проекте

Данный репозиторий создан в рамках учебной практики студентов 2 курса по направлению «UI-тестирование». В качестве объекта тестирования выбрано веб-приложение **AliExpress**.

Проект предполагает выполнение полного цикла тестирования, максимально приближенного к реальным условиям работы в IT-компании.

---

## Задачи проекта

- Тестирование приложения через **API**;
- Тестирование с воспроизведением действий пользователя (**UI-автотесты**);
- Написание документации к тестам.

## Стек технологий

| Инструмент | Назначение |
|------------|------------|
| **Java** | Язык программирования |
| **Maven** | Система сборки и управления зависимостями |
| **JUnit** | Фреймворк для модульного тестирования |
| **Selenide** / **Selenium** | Инструменты для UI-автоматизации |

---

## Этапы выполнения

1. Проектирование тестов (API + UI);
2. Согласование с заказчиком (преподавателем);
3. Написание автотестов и документации;
4. Сдача итогового результата.
